# Oca do Açaí - Cardápio Digital

Aplicativo de cardápio digital desenvolvido para a Oca do Açaí, como projeto da disciplina Prática Extensionista VIII.

## O que o app faz

- Login e cadastro de usuário
- Visualização do cardápio por categorias
- Adicionar itens ao carrinho
- Finalizar pedido

## Tecnologias

- Flutter / Dart
- GetIt (gerenciamento de dependências)

## Como rodar

```bash
flutter pub get
flutter run
```
